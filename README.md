# Учебный Telegram-бот на aiogram

## Описание проекта
Это учебный Telegram-бот, разработанный на Python с использованием библиотеки aiogram.

Бот демонстрирует классические базовые возможности Telegram-бота:
- работу с Reply-кнопками;
- работу с Inline-кнопками;
- обработку callback-запросов;
- динамическое изменение inline-клавиатуры;
- разбиение проекта на папки и файлы по классической структуре.

Проект выполнен в аккуратном учебном стиле и подходит:
- для практической работы;
- для домашнего задания;
- для портфолио начинающего разработчика;
- для дальнейшего расширения функционала.

---

## Реализованные возможности

### 1. Главное меню через Reply-кнопки

По команде `/start` бот показывает главное меню с двумя кнопками:
- Привет
- Пока

При нажатии:
- на кнопку `Привет` бот отвечает: `Привет, {имя пользователя}!`
- на кнопку `Пока` бот отвечает: `До свидания, {имя пользователя}!`

---

### 2. Inline-кнопки со ссылками

По команде `/links` бот показывает три inline-кнопки:
- Новости
- Музыка
- Видео

Каждая кнопка открывает соответствующую ссылку в браузере или приложении.

---

### 3. Динамическая inline-клавиатура

По команде `/dynamic` бот показывает кнопку:
- Показать больше

После нажатия эта кнопка заменяется на:
- Опция 1
- Опция 2

После выбора одной из кнопок бот отправляет сообщение с текстом выбранной опции.

## Установка и запуск

- 1. Создать проект

Создайте папку проекта, например: 
C:\Users\ВашеИмя\PyCharmProjects\telegram_homework_bot

- 2. Открыть проект в PyCharm

Откройте папку проекта в PyCharm.

- 3. Установить зависимости

Откройте Терминал в PyCharm и выполните команду: 
pip install -r requirements.txt

- 4. Вставить токен

Откройте файл config.py и вставьте ваш токен от @BotFather в переменную TOKEN.

- 5. Запустить проект

Запустите файл main.py (Run стрелочка)

## Команды бота

- /start — открыть главное меню
- /links — показать inline-кнопки со ссылками
- /dynamic — показать динамическую клавиатуру

---

#### Как это выглядит для пользователя:
    - Команда /start
        - Показывается красивое стартовое сообщение и reply-клавиатура:
            - Привет
            - Пока
            - Команда /links

    - Показываются inline-кнопки:
            - 📰 Новости
            - 🎵 Музыка
            - 🎬 Видео

    - Команда /dynamic
    - Сначала показывается:
            - Показать больше
    - После нажатия:
            - Опция 1
            - Опция 2
